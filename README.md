# 💰 Expense Tracker

## 📌 Project Overview

This is **Project 2** of the DecodeLabs Python Programming Industrial Training Kit (Batch 2026).

The Expense Tracker allows users to enter multiple expense amounts, continuously calculates the total amount spent using an accumulator, and displays the final total when the user finishes entering expenses.

---

## 🚀 Features

- Accept multiple expense entries
- Running total calculation
- Uses Python loops and accumulators
- User-friendly interface
- Simple and beginner-friendly code

---

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python expense_tracker.py
```

or

```bash
python3 expense_tracker.py
```
